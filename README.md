Pro League App:
This is a single page react which renders some famous league around the world which are fetched from.It also uses react router to make it more easier to move arounds the components. 
[TheSportsDbApi](https://www.thesportsdb.com/api.php).

Then you can explore the leagues and see some details about the leagues like founding date,sports type,country,gender and some description about them.
You will also see some social buttons which will take you to their facebook,twitter or youtube accounts.
